# Multimodal Travel Behavior and Sustainable Mobility Evaluation

This repository contains a Jupyter notebook for an open-source machine learning workflow focused on multimodal travel behavior inference and sustainable mobility program evaluation.

## Contents

- `Scalable_Open_Source_ML_Multimodal_Travel_Behavior_Sustainable_Mobility.ipynb`: Main notebook for the analysis workflow.

## Topics

- Multimodal travel behavior inference
- Sustainable mobility program evaluation
- Machine learning for transportation data
- Open-source research workflow

## Getting started

Open the notebook in Jupyter Notebook, JupyterLab, or Google Colab and run the cells after installing the required Python packages used in the notebook.
